## Install

```sh
yarn
```

## Prepare connection string

Replace *YOUR_MONGODB_URI* for your own *MongoDb* connection string. For example, in case of local mongodb server:

```javascript
const uri = 'mongodb://localhost/test';
```

## Running

```sh
yarn start
```
